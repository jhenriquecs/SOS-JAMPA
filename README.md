# SOS-JAMPA

Aplicação Flask (blueprints) para denúncias ambientais de João Pessoa - PB.
Armazena dados em CSV e implementa permissões de admin/comum.

Como rodar:
1. python -m venv venv
2. source venv/bin/activate  # ou venv\Scripts\activate no Windows
3. pip install flask
4. python run.py

Pasta `data/` será criada automaticamente com CSVs.
Uploads armazenados em `static/uploads/`.
